SEMANTIC ELEMENTS

Overview

Semantic HTML elements helps in describing the purpose of the element and the type of content that is inside them to browser and developer in a more readable way.

Examples of Semantic Elements in HTML5
  1. Article
  2. Details
  3. Footer
  4. Header
  5. Main
  6. Mark
  7. Nav
  8. Section

Benefits of Sementic Elements

  • Better SEO
  • Improves accessibility
  • Ease in code development

Conclusion

Semantic elements helps in structuring the code, making it more readable and easier to manage.
